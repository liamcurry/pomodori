Pomodori
========

Pomodori is a simple [Pomodoro timer][pomodoro] written in Javascript with
[Mithril][mithril].

[pomodoro]: https://en.wikipedia.org/wiki/Pomodoro_Technique
[mithril]: http://lhorie.github.io/mithril/
